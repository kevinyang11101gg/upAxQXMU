## 前言

欢迎来到基于SSM的在线考试系统项目！该项目致力于为用户提供一个便捷、高效的在线考试平台，通过运用Spring、Spring MVC和MyBatis等主流技术框架，实现了一套完善的在线考试系统。以下是关于本项目的详细说明。

## 内容介绍

基于SSM的在线考试系统主要包括以下模块：考生管理、试题管理、考试管理、成绩管理以及系统管理。系统为考生提供了在线答题、成绩查询等功能；为教师提供了试题发布、成绩统计等功能；为管理员提供了用户管理、系统设置等功能。通过这些模块的有机组合，为用户带来良好的在线考试体验。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC，MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
